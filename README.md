# practical-react-demos
Practical React demo code for training sessions
https://github.com/jazeee/practical-react-demos

Based on `create-react-app`

## To Run:
1. Clone this repository
1. Install NodeJS
1. Run `npm install`
1. Run `npm start`
1. To test, run `npm run test`
